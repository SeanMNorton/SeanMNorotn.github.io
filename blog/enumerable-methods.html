A Brief Guide to Enumerable and .map


Enumerables are extremely handy in Ruby. They are used to iterate through a collection of objects. The dictionary definition of the word enumerate:

  1. to mention separately as if in counting; name one by one
  2. to ascertain the number of; count

We can use them much like we would use a for statement. There are quite a few different types of enumerables some of them being: each, select, inject, map.

The simplest of them being each. We can use each on an array to return each item in the array, like so.

[ 10, 11, 4 ].each { |e| puts e}

This Returns:

10
11
4

Here the enumerable takes each and puts them through the code to the right of the .each statement. This assigns a variable e to each item coming through and then puts them out one at a time.

The map Enumerable

The map enumerable is often referred to as collect. Let us, much like above, say you are using an array but this time you want to change the data and then collect/map it into a new array.

array_1 = [ 2, 4, 6]
array_2 = array_1.map {|x| * 2}
puts array_1.join(, ) #=> 2, 4, 6
puts array_2.join(, ) #=> 4, 8, 12

This takes the original array, array_1 and multiplies each item by two and maps it to a new array, array_2. There is nothing odd about it. ;)

Hopefully this helps explain enumerables, how and why to use them as well as the functions of the map enumerable.
