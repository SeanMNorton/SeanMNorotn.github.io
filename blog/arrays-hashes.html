Arrays vs Hashes

Feburary, 14th 2016

Arrays


An array are: ordered, integer-indexed collections of any object.

Lets break this down a little bit... So what this means this that an array is a collection or "list" of an object(string,integer,etc...) these objects are ordered using and integer-indexing, starting at 0. So the first value will be indexed at 0 the second 1, third 2 etc...

array_name = [0, "one", 2, "three"]

Hashes


Hashes are: a collection of unique keys and their values.

These are really similar to arrays, but lets first break down what the line above means. So a hash is a collection or "list" of keys, now these keys are very similar to the integer-index that arrays use, but for keys we can define them as anything, such as a string or integer. Then the values can be any object just like in arrays.

hash_name = { key1: "value1", key2: "value 2", 3: 3}

In this line of code the hash name comes first and then we set it equal to a hash with keys of , key1: key2:, 3: each key is assigned a value the values above are "value1", "value 2" and 3.

In previous versions of Ruby you will see hashes defined as follows:

hash_name = { :key1 => "value1", :key2 => "value 2", :3 => 3}

In recent updates they have created the short hand which I used originally, they both do the exact same thing, just one is easier to type.

VS

Both arrays and hashes come in handy. Where I am at with my knowledge of ruby, beginner, I would not say that one is better than the other. Figuring out which one to use just comes down to what you need it for. If you wanted a list of friends that you had, you could use an array.

friends_array=[Joe, Dylan, Jon, Sammy, Kate]

Lets say you wanted to store specific details of a friend and you needed to retrieve those details with a set key, you would use a hash.

joes_hash = { haircolor: "brown", eyecolor: "blue", trustworthy:"no" }

If you wanted to see if Joe was trust worthy you would put:

joes_hash[trustworthy:]

this would give you "no"
Boom know you know not to trust Joe and it was real easy to look up.
